## ESTRUCTURA ESTÁNDAR POR ARTÍCULO

```
1. **Intro Hook** (100 palabras)
   - Pregunta incómoda
   - Stat impresionante
   - Promesa clara

2. **Índice/TDC**
   - Jump links

3. **Secciones Principales** (1200-1800 palabras)
   - Subheadings claros
   - 3-5 puntos clave
   - Ejemplos reales
   - Imágenes/capturas

4. **CTA Intermedio**
   - "¿Quieres asesoría?" → WhatsApp

5. **Resumen/Conclusión**
   - Recap breve
   - Call to action final

6. **CTA Final**
   - Formulario contacto
   - "Solicita consulta"
```
